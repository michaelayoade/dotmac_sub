{# Confirmation Modal Component ============================ A reusable confirmation modal for destructive actions. Usage: 1. Include this component in your base template (once) 2. Dispatch 'confirm-action' event with details: $dispatch('confirm-action', { title: 'Delete Item', message: 'Are you sure you want to delete this item?', actionUrl: '/admin/items/123/delete', method: 'POST', // or 'DELETE' variant: 'danger' // 'danger', 'warning', or 'info' }) Variants: - danger: Red styling, for delete operations - warning: Orange/amber styling, for deactivate/archive operations - info: Blue styling, for general confirmations #}
